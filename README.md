# 🚀 marn-express-initializer

A simple project starter for the **Code Biruni** ecosystem by **Robius Sani**.
Quickly scaffold a **MERN (Express.js) backend** with a clean structure and essential setup.

---

## ⚡ Installation & Usage

Run directly using `npx`:

```bash
npx marn-express-initializer <your-project-name>
cd <your-project-name>
```
- setup env
```
npm run dev
```
